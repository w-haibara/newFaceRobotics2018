* espで受信するパケット構造について
駆動系を進める向きをx方向，y方向それぞれ3bitの数値で指定する．
3bitのうち最初のbitは符号ビットであり，0で負，1で正を表す．
残りの2bitの00から11までの数でx，y成分の絶対値を表す.
シュート機構のサーボとギヤボックスのギミックのオンオフにそれぞれ1bitづつ使う．
計8bitのバイナリを機体のespが受信してロボットが動作する

|1|2|3|4|5|6|７　　|８　　　　　|
|x成分|y成分|サーボ|ギヤボックス|

ex) 11100101 →　111(x成分:+3)　001(x成分:-1)　0(サーボ:オフ)　1(ギヤボックス:オン)
